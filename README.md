# TIL 소개

- **TIL** : Today I Learned!
- 방향성이 잡힐 때까지는 교육에서 배운 것 정리하는 용도

## TIL_CLI

- **CLI** : Command Line Interface

- **Bash**에서 사용할 수 있는 기능들 간단 정리



## TIL_python

- **Python** 공부 내용
- 많이 사용해 본 언어인만큼, 전체정리보다는 몰랐던 내용, 헷갈렸던 내용 위주로 정리함



### 1급 객체의 조건

1. 변수에 할당 가능하다.
2. 인자로 넘길 수 있다.
3. 리턴할 수 있다.



## TIL_git

- **Git** 공부 내용
- 내용이 아주 다양하지는 않으므로 하나의 파일로 관리
